# Zig Compiler (zig)

Installs the Zig programming language (<https://ziglang.org/>) globally.

## Example Usage

```json
"features": {
    "ghcr.io/savvy-web/zig:0": {}
}
```

## Options

| Options Id | Description | Type | Default Value |
| ----- | ----- | ----- | ----- |
| zigVersion | Zig version to install (e.g., 0.12.0, master). | string | 0.12.0 |
