# Node.js + pnpm (strict)

Installs Node.js and pnpm with strict, reproducible versions for CI and Codespaces.

## Options

- `nodeVersion`: Node.js version (absolute, no semver ranges). Default: `24.11.0`
- `pnpmVersion`: pnpm version (absolute, no semver ranges). Default: `10.20.0`

## Usage

Add this feature to your `devcontainer.json` to install Node.js and pnpm.
